# Simple PHP CRUD Application

A simple CRUD (Create, Read, Update, Delete) application built 

## Requirements

- PHP 8.x
- VS Code

## Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/yourusername/simple-html-crud.git
```

### Step 2: Move the Project

Copy the project to:

```
C:\xampp\htdocs\
```

The project path should be:

```http://localhost/simple-crud-design-/
```

### Step 3: Start XAMPP

Start:

- Apache
